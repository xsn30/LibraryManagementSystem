# Library Management System

This project is a Java-based library management system developed as a learning project.

## Overview

The goal of this project is to practice object-oriented programming and basic system design by implementing a simple library management application.

## Implemented Features

- Basic library management functionality
- Book borrowing and returning
- Management of book-related records
- Java-based project structure for handling library operations

## Current Status

This project is still a work in progress.  
Some advanced features, such as barcode scanner integration and student card handling, are not fully completed yet.

## Technologies

- Java
- Object-Oriented Programming

## Notes

This project was originally developed for a Chinese-language environment, so some interface text or comments may appear in Chinese.
